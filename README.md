# Portfólio — Laura Lopes

Versão entregue: portfólio pronto em HTML/CSS/JS (mobile-first, tema dark, tipografia e copy prontos).

Arquivos principais
- `index.html` — estrutura, seções e SEO básico.
- `css/style.css` — estilos mobile-first, dark theme, tipografia (Syne + DM Sans) e animações.
- `js/script.js` — menu hambúrguer, smooth scroll, Intersection Observer para animações e botão WhatsApp.

Como abrir localmente
1. Coloque a pasta em um servidor local (ex.: `c:\xampp\htdocs\lauralopes-portfolio`).
2. Abra no navegador: `http://localhost/lauralopes-portfolio/` ou abra `index.html` diretamente.

Personalizações rápidas
- Trocar número do WhatsApp: abra `js/script.js` e atualize a constante `WA_NUMBER` com seu DDI+DNN (ex: `5511999999999`).
- Alterar textos/portfólio: edite as seções diretamente em `index.html`.

Testes recomendados
- Verifique responsividade em 320px, 768px e 1200px.
- Teste o botão principal e o FAB do WhatsApp (abre em nova aba).
- Verifique animações de entrada ao rolar a página.

Observações
- Todos os ícones e imagens estão embutidos como SVG data-URL para facilitar entrega sem ativos externos.
- Não use frameworks externos — projeto feito com HTML, CSS e JS puros.

Se quiser, posso:
- Gerar uma versão com formulário de contato integrado ao WhatsApp com campos (nome, telefone, mensagem).
- Criar um deploy simples usando GitHub Pages ou enviar um zip pronto para publicação.
