Sistema Lifebios - Hemograma automatizado

O Lifebios é um sistema automatizado para análise de hemogramas, desenvolvido para otimizar exames laboratoriais, reduzir erros e agilizar diagnósticos. Construído com HTML, CSS, JavaScript e tecnologias backend escaláveis, permite a leitura e processamento automático de dados, integrando-se a equipamentos laboratoriais. Destaca-se pela validação automática de resultados, alertas para valores críticos e geração de laudos personalizados, proporcionando mais eficiência e precisão na rotina laboratorial.

Aplicativo Progamer

O Progamer é um app voltado para gamers, oferecendo ferramentas para análise de desempenho, organização de partidas e integração com plataformas como Steam e Xbox Live. Desenvolvido com Flutter, conta com funcionalidades como estatísticas de jogo, agenda de torneios, tutoriais e matchmaking para conectar jogadores de nível semelhante. Com uma interface moderna e intuitiva, o app auxilia na evolução dos usuários no cenário gamer, promovendo aprendizado e interação.

Landing Page - Construtora Minatto

A Landing Page da Construtora Minatto foi desenvolvida com HTML, CSS e JavaScript, focada em apresentar os serviços da empresa de forma profissional e responsiva. O design transmite credibilidade e sofisticação, com animações sutis e elementos gráficos modernos. O site conta com seções estratégicas, como apresentação da empresa, serviços, portfólio, depoimentos e formulário de contato. Recursos como efeitos de rolagem, botão de WhatsApp fixo e carrossel de projetos garantem uma navegação dinâmica e conversões mais eficientes.
